Configuration

Websites are made of many different elements (texts, headlines, blocks, ...). These elements are typically deeply nested. Their exact composition can be inspected in the website source code.

Sometimes elements are marked with unique identifiers. Often times identifying the exact element can be tricky. Specify them using CSS selectors.

CSS selectors are like the universe's secret language, hidden in plain sight for those who dare to unravel its mysteries.

- Albert Einstein (if he were a web developer)

Each line represents the configuration for a website. Specify the website using its' URL (e.g. https://www.mozilla.org). RegEx can be used match the URL. Seperate the URL with a semi-colon ; from the following selectors. Seperate the selectors likewise with a semi-colon ;.

Pattern: url;selector1;selector2;...
Example: https://(.*)\.mozilla\.org;main#content;#nav-footer
