Content Focus

Readme

Websites are made of many different elements (texts, headlines, blocks, ...). These elements are typically deeply nested. Their exact composition can be inspected in the website source code. Sometimes elements are marked with unique identifiers. Often times selecting the exact element can be tricky. They can be specified by CSS selectors. The website inspector of any Browser's developer mode can be used to determine and verify the selectors.

CSS selectors are like the universe's secret language, hidden in plain sight for those who dare to unravel its mysteries.

- Albert Einstein (if he were a web developer)

In the section below, each line represents the configuration for a website. Specify the website using its' URL (e.g. https://www.mozilla.org). The specifier is interpreted as RegEx. All lines with specifiers matching the URL will be applied. Seperate the URL with a semi-colon ; from the following selectors. Seperate each selector likewise with a semi-colon ;.

Configuration

Pattern: url-regex;selector1;selector2;...
Example: https://(.*)\.mozilla\.org;main#content;.nav-footer
